Sobre Mim

Sou formada em Medicina Veterinária pela PUC Minas, porém acabei não seguindo pois me deseludi com a prática da profissão e o quão ela é desvaloriada. A partir disso, decidi buscar outras alternativas de trabalho e sempre tive o interesse em saber como seria a área de desenvolvimento. Depois de muita reflexão achei minha vocação na área Frontend, visto que sempre gostei de elementos visuais e de tecnologia.
Mesmo não seguindo a profissão veterinária, essa área me ensinou muito disciplina, determinação e trabalho em grupo, sempre levarei esses ensinamentos na minha vida.
Sou uma clássica nerd/geek, como você preferir chamar, gosto de jogar jogos bonitinhos e casuais, animes, filmes (principalmente de animação) e séries. Sou muito feliz em fazer parte da comunidade de desenvolvimento e apesar de ainda não ter exercido a função, tenho certeza que me agarrarei à oportunidade que surgir e farei o meu melhor.

Meus Projetos

  • FoodDev

    Link do projeto

    O projeto é sobre um site para um restaurante fictício, com um layout visualmente agradável, que permite o usuário visualizar o catálogo e realizar uma compra. Nesse projeto explorei os recursos do JavaScript, incluindo um carrossel de imagem, catálogo dinâmico e validação de formulário.

  • Weather Forecast

    Link do projeto

    Weather Forecast é um app de previsão do tempo, através campo de busca permite que o usuário busque a cidade cuja previsão do tempo se deseja consultar. Ao clicar no botão Buscar ou na tecla Enter, o sistema faz uma busca utilizando a API do Open Weather API.

  • Github API

    Link do projeto

    O projeto é o resultado de um desafio do curso de desenvolvimento front-end DevQuest do módulo de JavaScript avançado. Através do campo de busca deve inserir o nome do usuário cujo perfil deseja consultar, ao clicar no botão consultar ou apertar a tecla 'Enter',, o sistema faz a busca através da API do GitHub.

  • TravelNow

    Link do projeto

    TravelNow é uma landing page de uma agência de viagens. Esse projeto foi desenvolvido com HTML, CSS e layout responsivo. Grid e Flexbox.

  • Loja de Roupas

    Link do projeto

    Site de uma loja de roupas virtual que contém vários elementos em conjunto com mais de uma página, contando com uma homepage, um catálogo e uma página de vendas, todas integradas entre si e de forma responsiva. Utilizei HTML, SASS e Bootstrap para desenvolver o projeto.

  • Validação de Formulário

    Link do projeto

    Esse projeto foi um desafio proporcionado pelo curso de desenvolvimento front-end DevQuest, o objetivo era recriar um formulário fornecido pelo Figma, fazer a validação do mesmo com JavaScript. Ao clicar no botão, caso o usuário não tenha preenshido corretamente, o campo ficará vermelho e aparecerá uma mensagem, caso contrário o campo ficará com as bordas verder.
