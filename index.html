Sobre Mim

Durante minha vida, vi pessoas que admiro profissionalmente seguindo o caminho da tecnologia e do desenvolvimento, e eu por ser estudiosa e inquieta (no bom sentido), resolvi me debruçar no desenvolvimento front-end.
Comecei de forma auto-didata e logo percebi que tenho habilidade para a área e busquei um curso profissionalizante para realmente me aprofundar, entender os fundamentos, as boas práticas, do básico ao mais avançado, construindo um conhecimento sólido nas stacks que englobam o front-end.
Meu objetivo é avançar cada vez mais na carreira, aperfeiçoando minhas habilidades e soft-skills, me mantendo atualizada e aprender mais tecnologias para ser uma profissional de qualidade e futuramente me tornar full-stack. Atualmente estou aprendendo React e testes automatizados com Jest.
Sou uma clássica nerd/geek, como você preferir chamar, gosto de jogar jogos bonitinhos e casuais, animes, filmes (principalmente de animação) e séries. Sou muito feliz em fazer parte da comunidade de desenvolvimento e apesar de ainda não ter exercido a função, tenho certeza que me agarrarei à oportunidade que surgir e farei o meu melhor.

Meus Projetos

  • FoodDev

    Link do projeto

    O projeto é sobre um site para um restaurante fictício, com um layout visualmente agradável, que permite o usuário visualizar o catálogo e realizar uma compra. Nesse projeto explorei os recursos do JavaScript, incluindo um carrossel de imagem, catálogo dinâmico e validação de formulário.

  • Weather Forecast

    Link do projeto

    Weather Forecast é um app de previsão do tempo, através campo de busca permite que o usuário busque a cidade cuja previsão do tempo se deseja consultar. Ao clicar no botão Buscar ou na tecla Enter, o sistema faz uma busca utilizando a API do Open Weather API.

  • Github API

    Link do projeto

    O projeto é o resultado de um desafio do curso de desenvolvimento front-end DevQuest do módulo de JavaScript avançado. Através do campo de busca deve inserir o nome do usuário cujo perfil deseja consultar, ao clicar no botão consultar ou apertar a tecla 'Enter',, o sistema faz a busca através da API do GitHub.

  • TravelNow

    Link do projeto

    TravelNow é uma landing page de uma agência de viagens. Esse projeto foi desenvolvido com HTML, CSS e layout responsivo. Grid e Flexbox.

  • ScienceBlog

    Link do projeto

    Blog fictício sobre notícias de ciência no geral. Esse projeto teve como objetivo aplicar em prática as bibliotecas do React.js, as quais foram react-router-dom e Styled Components. Além disso, as notícias foram criadas dentro de um arquivo JSON e posteriormente chamadas para a aplicação como se fossem uma API.

  • Validação de Formulário
    com React.js

    Link do projeto

    O projeto foi feito usando React e a biblioteca React-Hook-Form, com o intuito de criar um formulário da maneira mais simplificada, dessa forma deixando-o iterativo. Este projeto inclui um formulário simples com campos para o nome, sobrenome, e-mail e senha. Ele demonstra como usar os hooks do React Hook Form para gerenciar o estado do formulário, lidar com a validação dos campos e lidar com a submissão do formulário Além disso, apresenta responsividade a todos os tamanhos de tela.
